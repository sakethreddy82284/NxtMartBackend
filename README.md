# NxtMart Backend

The powerful MERN engine powering the NxtMart multi-role ecosystem.

## 🚀 Role-Based Features
- **Customer:** Secure cart, real-time order placement, and history.
- **Store Manager:** Category CRUD and dynamic order-to-partner assignment.
- **Delivery Partner:** Task-based dashboard with status updates.
- **Admin:** Global platform monitoring and user role management.

## 🛠️ Tech Stack
- **Node.js & Express:** Robust API architecture.
- **MongoDB & Mongoose:** Scalable document storage.
- **JWT & Cookies:** Secure, role-based authentication.

## ⚙️ Setup
1. `npm install`
2. Create `.env` with `PORT`, `JWT_SECRET`, `DB_USER`, `DB_PASSWORD`.
3. `npm run dev`
